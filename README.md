# THE3 Studio 🚀

THE3 Studio의 웹사이트 및 관리자 대시보드 프로젝트입니다.

## 🔗 접속 주소 (Links)

### [로컬 개발 환경]
*   **Main Site:** [http://localhost:8080](http://localhost:8080)
*   **Admin Dashboard:** [http://localhost:8080/admin/](http://localhost:8080/admin/)

### [배포 환경 (Vercel)]
*   **Vercel Live:** [https://the-3-wep.vercel.app](https://the-3-wep.vercel.app)
*   **Vercel Admin:** [https://the-3-wep.vercel.app/admin/](https://the-3-wep.vercel.app/admin/)

---

## 🛠 주요 기능
- **메인 히어로 영역:** 텍스트 및 미디어 동적 관리 (Firebase Storage 연동)
- **포트폴리오:** 작업물 추가, 수정, 삭제 및 카테고리 분류
- **클라이언트 관리:** 협업사 로고 및 정보 관리
- **문의 내역:** 실시간 고객 문의 확인 및 관리

## 💻 설치 및 실행
1.  **Repository Clone:**
    ```bash
    git clone https://github.com/withtms-max/THE3WEP.git
    ```
2.  **PHP Server 실행 (로컬):**
    ```bash
    php -S localhost:8080
    ```
3.  **데이터베이스 초기화:**
    `db_init.php` 또는 `db_init_sqlite.php`를 실행하여 초기 설정을 진행하고, Firebase 설정을 확인합니다.
