## Project Overview
This project contains both the React Web version and the Flutter App version in a monorepo structure.

- **Web**: Root directory (`/`)
- **App**: Flutter directory (`/flutter_app`)

---


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
